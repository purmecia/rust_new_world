# Integer to Roman

## Usage
Run `cargo run -- integer-to-roman -i [Integer]` to change the integer into roman number!
Like `cargo run -- integer-to-roman -i 19` will output `XIX`
Like `cargo run -- integer-to-roman -i 100` will output `C`
